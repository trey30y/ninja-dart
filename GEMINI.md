## 🚀 Quick Start Commands

- **Format code:** `Cmd + S` (Save) or `Shift + Option + F`
- **Fix Syntax:** `Cmd + .` (Quick Fix for semicolons)

## 🧠 Personal Cheat Sheet

### Variables
- `var`: Type is inferred but locked once set.
- `final`: Cannot be changed once assigned (Runtime).
- `const`: Cannot be changed (Compile-time).

### Collection Shortcuts
- **List:** `['item1', 'item2']`
- **Map:** `{'key': 'value'}`

## 🛠 Troubleshooting Notes
- **Formatter not working?** Check the "Problems" tab for missing `;`.
- **Terminal error?** Ensure you are inside the correct folder (`cd folder_name`).
